  ---
  Request flows

  Upload flow

  Client → main.py → parser.py (decode + clean DF)
                   → session_store.py (create session, store DFs)
                   → schema_analyzer.py (stats + starter questions)
                   ← UploadResponse (session_id, schema, questions)

  Query flow

  Client → main.py → session_store.py (get session + DFs)
                   → claude_service.py (system prompt + history → Bedrock)
                   → code_executor.py (AST check → subprocess → result)
                   → _inject_result_value() (replace RESULT_VALUE placeholder)
                   → session_store.py (append to history)
                   ← QueryResponse (chat_message, artifact HTML, insight)

  ---
  Key design decisions

  ┌─────────────────────────────┬────────────────────────────────────────────────────────────────────┐
  │          Decision           │                               Reason                               │
  ├─────────────────────────────┼────────────────────────────────────────────────────────────────────┤
  │ In-memory sessions          │ MVP simplicity — no DB dependency                                  │
  ├─────────────────────────────┼────────────────────────────────────────────────────────────────────┤
  │ Subprocess execution        │ Isolates user code from server process                             │
  ├─────────────────────────────┼────────────────────────────────────────────────────────────────────┤
  │ AST validation before exec  │ Blocks dangerous ops before subprocess even spawns                 │
  ├─────────────────────────────┼────────────────────────────────────────────────────────────────────┤
  │ RESULT_VALUE placeholder    │ Claude never invents numbers — real value injected after execution │
  ├─────────────────────────────┼────────────────────────────────────────────────────────────────────┤
  │ Schema injected into prompt │ Claude knows column names, types, stats without seeing raw data    │
  ├─────────────────────────────┼────────────────────────────────────────────────────────────────────┤
  │ Conversation history        │ Enables follow-up questions with full context                      │
  ├─────────────────────────────┼────────────────────────────────────────────────────────────────────┤
  │ chardet + multi-delimiter   │ Handles real-world messy CSV files reliably                        │
  └─────────────────────────────┴────────────────────────────────────────────────────────────────────┘
